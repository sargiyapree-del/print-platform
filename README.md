# 🖨️ Print Service Platform

A full-stack web application that allows users to upload documents, place print orders, and schedule pickup times — with an admin panel for managing orders.

---

## 🚀 Features

### 👤 User Features
- Upload documents (PDF, Word)
- Select number of copies
- Automatic price calculation
- Schedule pickup date & time
- View order history
- Real-time order status updates

### 👨‍💼 Admin Features
- View all orders
- Update order status (pending → completed)
- Manage full order lifecycle

---

### Backend
